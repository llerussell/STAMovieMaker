## Supported file formats
Movies:
* TIFF
* BIN (Bruker (custom format, [see here](https://github.com/llerussell/Bruker_PrairieLink)))
* RAW (Thorlabs)

Stimulus times:
* PackIO
* ThorSync
* Txt files
